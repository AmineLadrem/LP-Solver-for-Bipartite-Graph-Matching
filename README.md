# LP Solvers for Bipartite Graph Matching

This project contains experiments for LP solvers on bipartite graph matching.

## Run the Experiment

Open the `.ipynb` notebook file and run all cells.

You can run it using Jupyter Notebook or JupyterLab:

```bash
jupyter notebook
```

or

```bash
jupyter lab
```

Then open the notebook and execute the cells from top to bottom.
